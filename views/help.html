Instructions

The objective of this challenge is to access this website via "Representational State Transfer", more commonly referred to as "REST".

REST is useful to master, since a lot of websites use it to change the state of their website.

What you have to do

Write a C# program capable of doing GET (Read), PUT (Update), and DELETE calls on the server to modify the database.
The database will display entries on the front page.

Your REST Entry Points

TypeURIDescription
GET/listGets the whole table as a JSON array
GET/list?sig=:q:Gets a part of the table where the signature matches the query :q:
PUT/list/:m:/:s:Adds a new message to the table. :m: is the message, :s: is the name of the person who wrote it
DELETE/list/:i:Deletes an entry from the table given by its ID :i:
